# 히어로 슬라이드 이미지

여기에 홈 첫 화면 슬라이드 이미지를 넣으세요 (예: `hero1.jpg`, `hero2.jpg`, `hero3.jpg`).

권장 사양
- 가로로 넓은 이미지 (가로:세로 ≈ 16:9 또는 더 와이드)
- 최소 1600px 폭 이상 (선명하게)
- JPG 권장, 한 장당 300~500KB 이하로 최적화하면 로딩이 빠릅니다

적용 방법: `index.html` 의 각 `.slide` 에서
`style="background-image: linear-gradient(...)"` 부분을
`style="background-image: url('assets/img/hero1.jpg')"` 처럼 바꾸면 됩니다.
